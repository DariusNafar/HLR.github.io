HetSaul

This project augments Saul to support working with heterogeneous and multimodal information, e.g., vision and language. Saul is modeling language implemented as a domain specific language (DSL) in Scala. The main goal of Saul is to facilitate designing machine learning models with arbitrary configurations for the application programmer.
